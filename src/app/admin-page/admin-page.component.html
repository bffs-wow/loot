Manual TMB Import

Note: this should not need to be done normally, only if the auto-import process is not working, or there is a desire for an immediate TMB data update.
  1. Open these pages:
    • https://thatsmybis.com/19497/best-friends/export/characters-with-items/html
    • {{ tmbEditPage }}
  2. Copy ALL of the text from the ThatsMyBis page and paste it into the github page 'Edit file' text box, overwriting the existing contents.
  3. Press the green 'Commit changes' button.
  4. That's it! It may take a few minutes for the change to appear on the site after reloading.
